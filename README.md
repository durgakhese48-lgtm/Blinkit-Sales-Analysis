# Blinkit Sales Analysis Dashboard

## Overview

This project is an interactive Power BI dashboard developed to analyze Blinkit's sales performance, customer behavior, and business insights using data visualization techniques.

The dashboard provides meaningful insights into sales trends, product categories, outlet performance, and customer purchasing patterns to support data-driven decision-making.

---

# Features

- Interactive Sales Dashboard
- KPI Analysis
- Product Category Insights
- Outlet Performance Tracking
- Customer Purchase Analysis
- Dynamic Filters and Visualizations
- Business Intelligence Reporting

---

# Tools & Technologies

- Power BI
- Data Visualization
- Business Analytics
- Dashboard Design
- Data Cleaning

# Dashboard Insights
